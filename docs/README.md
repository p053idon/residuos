# EcoPort Dashboard (React + Vite)

Scaffold React + Vite para o MVP do sistema de gestão de resíduos portuários.

## Como rodar local

1. Instale dependências
```bash
npm install
```

2. Rodar em dev
```bash
npm run dev
```

3. Build
```bash
npm run build
npm run preview
```

## Deploy no GitHub

- Crie repositório no GitHub e faça push.
- Use Vercel/Netlify para deploy automático (mais simples).
- Para GitHub Pages é preciso gerar build e publicar `dist` (recomendo Vercel).

## Observações
- Frontend pronto para integrar com Supabase ou API backend.
